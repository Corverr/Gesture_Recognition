# Распознавание жестов русского жестового языка по видео

## Данный проект посвящен разработке и сравнительному анализу моделей глубокого обучения для распознавания динамических жестов русского жестового языка (РЖЯ) на основе видеоданных. Проект реализован в рамках учебной практики и охватывает полный цикл разработки решения в области компьютерного зрения: от постановки задачи и анализа данных до обучения моделей и оценки результатов.
